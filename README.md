# markdown2html
For personal usage:
markdown2html with **dir**   
>`node ./build/markdown2html.js {markdowndir} {htmldir}`

# Build 
## install modules
>`npm install` or `yarn install`
## compile typescript 
>`tsc`   

compile `src/app.ts` to `dist/app.js`
## pack
>`npx webpack`

pack all module into a single js file : `build/markdown2html.js`.
